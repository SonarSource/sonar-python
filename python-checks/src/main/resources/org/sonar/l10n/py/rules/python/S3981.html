Why is this an issue?

The length of a collection is always greater than or equal to zero. So testing that a length is greater than or equal to zero doesnâ€™t make sense, since the result is always true. Similarly testing that it is less than zero will always return false. Perhaps the intent was to check the non-emptiness of the collection instead.

Noncompliant code example

mylist = []
if len(myList) >= 0:  # Noncompliant
    pass

if len(myList) < 0:  # Noncompliant
    pass

Compliant solution

mylist = []
if len(myList) >= 42:
    pass

if len(myList) == 0:
    pass
