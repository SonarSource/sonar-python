__all__ is used to define the list of module's names that should be imported when from package import * is used. For that reason, it may only contain strings.

Noncompliant Code Example

class MyClass:
    pass

__all__ = [
    MyClass  # Noncompliant
]

Compliant Solution

class MyClass:
    pass

__all__ = [
    "MyClass"
]
